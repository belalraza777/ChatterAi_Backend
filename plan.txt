Project Plan: ChatGPT-like Bot using EJS, Node.js, Express, MongoDB

GOAL:
Build a chatbot web app where users can:
- Register and log in
- Start new chat threads
- Send and receive messages using GPT-3.5
- View and continue previous chat threads

TECH STACK:
- Frontend: EJS, HTML, CSS (Bootstrap or Tailwind)
- Backend: Node.js, Express.js
- Database: MongoDB (Mongoose)
- AI: OpenAI API (Gpt 4.1)
- Auth: JWT 


STEP-BY-STEP PLAN:

1. Setup Project
- Initialize Node.js project
- Install all required dependencies
- Setup EJS and static folder

2. Database Models
- Create separate models for User, ChatThread, and Message
- Messages are stored in a separate collection

3. MongoDB Connection
- Setup MongoDB connection using Mongoose
- Use dotenv to manage environment variables

4. Authentication (Optional)
- Setup user registration and login using Passport.js
- Hash passwords using bcrypt
- Store sessions using express-session and connect-mongo

5. Chat Logic
- Create routes to:
  - View all chat threads
  - Create a new thread
  - View a single thread with message history
  - Post a message and get GPT reply

6. OpenAI Integration
- Use OpenAI API to generate responses
- Store both user and assistant messages in DB
- Keep full message history per thread

7. EJS Frontend
- Build layout with sidebar for chat threads
- Display messages in chat window
- Add form to send new messages

8. Deployment (Optional)
- Deploy backend on Render or Railway
- Use MongoDB Atlas
- Deploy frontend on same server or static hosting

EXTRA FEATURES (Future Scope)
- Auto-generate titles for threads using GPT
- Add chat deletion
- Add voice-to-text input
- Allow PDF upload for AI to read

END OF PLAN.

5. Routes Overview
HTTP Method	Route	Purpose
POST	/api/user/signup	Register new user
POST	/api/user/login	User login
GET	/api/user/logout	User logout

GET	/chat	Display all chat threads for user
POST	/chat/new	Create a new chat thread
GET	/chat/:threadId	Show messages in a specific thread
POST	/chat/:threadId/message	Send message & get GPT reply
